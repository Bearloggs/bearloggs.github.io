## Getting Started with Astro

Astro is a modern static site generator that makes it easy to build fast websites. Let's explore some of its key features.

### Installation

To get started with Astro, you can use the following command:

```bash
npm create astro@latest
```

### Project Structure

Astro projects follow a conventional structure:

```
/
├── public/
├── src/
│   ├── components/
│   ├── layouts/
│   └── pages/
└── astro.config.mjs
```

## Features and Benefits

### Performance

Astro ships zero JavaScript by default, making your sites incredibly fast.

### Component Islands

Astro's component islands architecture allows you to use any UI framework you want.

### Content Collections

Content collections make it easy to manage your content with type safety.
